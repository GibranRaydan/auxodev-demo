# Simple Django Rest Framework example

This is an example of django rest framework back-end with pytest

### How to Install 🔧

```
pip install django 
```
```
pip install djangorestframework 
```
```
pip install pytest 
```
```
pip install pytest-django
```

## Testing

_Just Run pytest like:_

```
pytest
```
